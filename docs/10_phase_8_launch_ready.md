# 10 - Phase 8: Launch Ready

## Goal

Move from MVP to a trustworthy public launch.

## Launch Content Target

Better:
- 5 excellent topics
- 10-20 strong collection pages
- 50-150 genuinely good printables

Not:
- 500 mediocre AI images

Quality wins.

## Additions

- Pagefind search
- simple filters
- Pinterest-friendly social images
- print CSS
- accessibility pass
- content quality checklist
- download click tracking plan
- legal/trust page review

## Task 8.1: Pagefind Search

Add after templates are stable.

Fields:
- title
- content type
- hub
- topic
- tags
- audience
- difficulty
- printable type

## Task 8.2: Simple Filters

Topic page filters:
- All
- Easy
- Detailed
- Classroom
- Seasonal
- type-specific tags

Use simple client-side JS only if needed.

## Task 8.3: Print CSS

Add `@media print` styles:
- hide nav/footer/buttons
- make printable page preview printable if relevant
- do not print ads
- do not print background decorations

## Task 8.4: Accessibility Pass

Check:
- color contrast
- keyboard focus
- alt text
- button labels
- aria labels where needed
- no hover-only actions
- large tap targets

## Task 8.5: Content Quality Checklist

Create:
`docs/content_quality_checklist.md`

Include:
- copyright check
- AI artifact check
- print quality check
- metadata check
- SEO check
- preview check

## Phase 8 Done Criteria

- site feels public-ready
- content quality is high
- search works
- accessibility basic pass complete
- print CSS exists
