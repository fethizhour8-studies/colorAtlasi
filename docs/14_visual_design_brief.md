# 14 - Visual Design Brief

## Design Objective

Create a warm, colorful printable catalogue.

The site should make users feel:
- this is easy
- this is safe
- this is useful
- I can quickly find something to print
- this is not a spammy AI site

## What To Avoid

- giant black SaaS hero
- sterile all-white design
- chaotic children’s site
- too many gradients
- complicated menus
- language dropdowns in MVP
- tiny thumbnails
- plain white printable previews on white cards

## Layout Personality

Use:
- rounded cards
- warm cream backgrounds
- pastel panels
- visible paper frames
- subtle shadows
- friendly badges
- clear CTA buttons

## Homepage Layout

Above the fold:
- compact brand message
- category cards or featured collections
- preview cards visible quickly

Hero should not dominate the page.

## Card System

Printable preview card should feel like:
- a folder
- a paper sheet
- a printable product
- organized and trustworthy

Not:
- an ad
- a random image
- a plain document

## Button Hierarchy

Primary:
- Download PDF

Secondary:
- Print
- View Page
- Browse Collection

Avoid:
- "Click here"
- buttons that look like ads
- too many buttons

## Mobile

Mobile is very important.

Rules:
- readable cards
- tap targets large
- no hover-only actions
- nav must not break
- printables visible quickly
- avoid complex dropdowns
