# 15 - Quality Checklists

## Printable Quality Checklist

Before publishing:
- no protected character/franchise
- no logo/trademark
- clean line art
- printable at 300 DPI target
- no weird AI artifacts
- no broken hands/objects
- no illegible worksheet text
- white background
- safe margins
- PDF opens
- preview looks good
- thumbnail readable
- metadata complete
- humanReviewed true

## Page Quality Checklist

Before indexable:
- useful intro
- no filler
- no keyword stuffing
- unique title
- unique meta description
- canonical correct
- robots correct
- breadcrumbs
- related collections
- FAQ only if useful
- no broken downloads
- no copyrighted topic
- clear license note

## UX Checklist

- printables appear early
- grid is easy to browse
- download buttons are obvious
- cards do not look like ads
- no modal/lightbox
- no full-pack button in MVP
- no language dropdown
- mobile works
- nav does not break

## Security Checklist

- only `dist/` deployed
- no `.env`
- no source repo
- no pipeline folder public
- no raw source images public
- no manifests public
- Cloudflare on
- directory listing off
- backups exist
